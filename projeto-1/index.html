Noções Básicas de HTML

O início dos computadores

Os computadores foram inicialmente criados para fazer cálculo. O primeiro computador da história foi o ENIAC. O ENIAC começou a ser desenvolvido em 1943, mas só entrou em funcionamento em 1946, após a segunda guerra mundial. Este computador era usado para cálculos balísticos, tendo uma alta velocidade de cálculos para a época.

Client Side x Server Side

Basicamente explicando, existem 2 lados na programação: Client Side (FrontEnd) e Server Side (BackEnd). Client Side é o lado do requisitor, ou seja, o dispositivo que está "pedindo" algo para o servidor. Já o Server Side é o "garçom", que serve o Client na medida do que foi requisitado. Existem linguagens de programação que atuam em cada lado. No Client Side existe HTML, CSS e JavaScript. Já no lado Server Side existem inúmeras outras, mas uma delas se destaca por atuar nos dois lados: o JavaScript. No Server Side o JavaScript necessita de uma ferramenta chamada Node.js, que atua juntamente com a engine V8 dos navegadores.

O que é HTML

HTML significa Hyper Text Markup Language, ou seja, linguagem de marcação hiper texto. Essa ferramenta não é uma linguagem de programação. Podemos dizer que o HTML é o esqueleto do site, toda estrutura está nele. É necessário o uso de navegadores para "ler" o HTML. É também necessário seguir alguns padrões pré estabelecidos, estes padrões podem ser encontrados no site da W3C, juntamente com as tags, assunto do próximo tópico.

Tags

As tags são usadas para estruturar o HTML. Os navegadores interpretam as tags de maneira que nos mostrem visualmente o que elas significam. Existem muitas tags no HTML, alguns exemplos são as tags de texto:

  • p: usada para parágrafos
  • h1 ao h6: usada para títulos e subtítulos
  • strong: usada para deixar um texto ou palavra em negrito
  • i: usada para deixar um texto ou palavra em itálico

Essas são apenas algumas das inúmeras tags do nosso tão importante e amado HTML.
